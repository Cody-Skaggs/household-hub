# Household Hub

A shared household organization app for multifamily homes, roommates, and smaller families.  
The goal is to provide a **centralized hub** for schedules and events, starting with a shared real-time calendar.

## Status

This project is under active development as a learning and portfolio project.

Current focus:

- Defining the product and data model.
- Designing the Supabase schema.
- Setting up the Android app (Kotlin + Jetpack Compose) to talk to Supabase.

## Tech Stack

- **Mobile:** Android, Kotlin, Jetpack Compose
- **Backend:** Supabase (Postgres, Auth, Realtime)
